# Lab Sheet 04 – Full Website Development with HTML, CSS, and JavaScript

**Module:** SE2011 – Technical Evolution of Multimedia  
**Topic:** Full Website Development with HTML, CSS, and JavaScript  
**Duration:** 2 Hours  
**Type:** Self-guided lab sheet  
**Output:** A fully functional website using HTML, CSS, and JavaScript.

---

## 1. Lab Objective

In this lab, you will create a **fully functional website** for **Campus Tech Store**. You will use:

- **HTML** to build the structure of the website
- **CSS** to style the website and make it visually appealing
- **JavaScript** to add interactivity to the website

Your goal is to create a **Campus Tech Store** website from scratch, which integrates HTML, CSS, and JavaScript.

---

## 2. Project Setup

### Step 1: Create the Project Folder

1. Create a folder named `IWT_Lab_04` on your computer.

2. Inside `IWT_Lab_04`, create the following subfolders:

    * `images` (for your logo and hero image)
    
    * `styles` (for your CSS file)
    
    * `js` (for your JavaScript file)
    
    * `src` (for your HTML file)

The folder structure should look like this:

```text
IWT_Lab_04/
│
├── src/
│   ├── index.html
├── images/
│   ├── logo.png
│   └── hero.jpg
├── styles/
│   └── styles.css
└── js/
    └── script.js

```

## 3. Step 2: Create the `index.html` File

1. Inside the `src` folder, **create a file called `index.html`**.

2. This file will be the main HTML file for your website.

3. Use HTML tags to structure your page with the following sections:

   * **Header**: Include the logo, title, and a short description.
   
   * **Navigation Bar**: Include links for the main sections of the page (e.g., Home, Products, Offers, About, Contact).
   
   * **Main Section**: Add sections for the homepage, product listing, offers, and contact information.
   
   * **Footer**: Display basic footer information with links to social media or external resources.

The sections you should have in `index.html` are:

* **Header**:

   * Include the store's logo and a title. The logo should be placed above the heading.
   
   * The title should be displayed in an `<h1>` tag.
   
* **Navigation Bar**:

   * Create links that navigate to different sections of the page. Use a navigation bar with a horizontal list of items.
   
* **Main Section**:

   * **Homepage Section**: Add a welcome message and a hero image.
   
   * **Products Section**: Add a table with product names, descriptions, and prices.
   
   * **Offers Section**: Display student offers in an ordered list.
   
   * **Contact Section**: Include a form to collect user details like name, email, and message.
   
* **Footer**:

   * Include basic footer information and links (e.g., copyright information).

---

## 4. Step 3: Create the `styles.css` File

1. Inside the `styles` folder, **create a file called `styles.css`**.

2. Use CSS to style your webpage, applying the following:

   * **Body Styles**:
       * Set a font family for the page and ensure that margins and paddings are set to zero for all elements.
       * Set a light background color for the page.
   
   * **Header**:
       * Set a background color for the header, and center-align the logo and title.
       * Set the logo to a specific width.
   
   * **Navigation Bar**:
       * Style the navigation bar to be horizontal. Set the background color to dark and text color to white.
       * Style the list items and links in the navigation bar.
       * Add hover effects on the links (e.g., change color when hovered over).
   
   * **Main Section**:
       * Style the main section to include padding and borders.
       * Style the product table, ensuring that the table is styled and easy to read.
   
   * **Footer**:
       * Center-align the footer and give it a background color.
       * Style the footer text.

---

## 5. Step 4: Create the `script.js` File

1. Inside the `js` folder, **create a file called `script.js`**.

2. Add JavaScript functionality to your page by following these steps:

   * **Welcome Message**:
       * Create a button that, when clicked, displays a welcome message below the button.
   
   * **Discount Button**:
       * Create a button to apply a discount. When clicked, display a discount message like "You received a 10% discount!".
   
   * **Form Submission**:
       * Add form validation. When the user submits the contact form, make sure the name and email fields are not empty.
       * Show a thank-you message below the form when submitted successfully.

---

## 6. Step 5: Testing Your Website (1 Hour)

1. **Open `index.html`** in your browser and test the following:

   * The **navigation bar** links should navigate to the appropriate sections of the page.
   * The **product table** should display correctly with product names, descriptions, and prices.
   * The **discount button** should display a discount message when clicked.
   * The **form** should validate user input and show a success message after submission.

2. **Test Responsiveness**:

   * Ensure that the webpage looks good on both desktop and mobile devices. Adjust your CSS to make the page responsive, if necessary.

---

## 7. Step 6: Final Task (Last 30 Minutes)

For the final task, make the following improvements to your website:

### Task A: Add an Interactive Product Search

1. Add a search input field above the product table.
2. When the user types a product name in the search field, dynamically filter the product table and display only matching products.

### Task B: Improve the Form with More Validation

1. Ensure that the user enters a valid email address.
2. If the email is invalid, show a message under the form asking the user to enter a valid email.

### Task C: Add Dark Mode Toggle

1. Add a button that allows users to toggle between **light** and **dark** mode on the website.
2. Use JavaScript to switch between dark and light themes by changing the background and text colors.

---

## 8. Submission

Once you have completed all the steps, save all your files and submit the following:

* `index.html`
* `styles/styles.css`
* `js/script.js`

Ensure that:

* All the buttons and interactions are working.
* The form validation is functional.
* The product table is displayed and searchable.
* Your website is styled according to the provided guidelines.
* The dark mode toggle is functioning as expected.
